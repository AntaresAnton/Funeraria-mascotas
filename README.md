# Huellas al Cielo - Mockup de Funeraria de Mascotas 🐾

Este proyecto es un **mockup funcional** y una propuesta de diseño web para una funeraria de mascotas, creado con un enfoque en la empatía, la claridad y una estética moderna.

## 🕯️ La Historia detrás del Proyecto

Este diseño nace de una experiencia personal profunda: la pérdida de mi querido gato, **Tomi**.

Durante el difícil proceso de buscar servicios funerarios para despedirlo, me encontré con que la mayoría de los sitios web del rubro eran poco intuitivos, estéticamente fríos o desactualizados, lo que dificultaba aún más la experiencia en un momento de duelo.

Decidí crear este proyecto como mi granito de arena, sirviendo como una referencia de cómo la tecnología y el buen diseño (UI/UX) pueden aportar dignidad y calidez a estos servicios esenciales.

## 🌟 Características del Diseño

*   **Estética "Soft & Serene"**: Uso de una paleta de colores calmada (verdes profundos, beige y acentos terracota) para transmitir paz.
*   **Enfoque UX**: Navegación clara para usuarios que están pasando por un momento emocional difícil.
*   **Componentes Funcionales**:
    *   Calculadora de presupuestos dinámica (JS) basada en especie y peso.
    *   Memorial Virtual interactivo.
    *   Catálogo de urnas con filtros.
*   **Atomic Design**: El proyecto incluye un archivo `atomic-design.html` que documenta los átomos, moléculas y organismos utilizados.
*   **Recursos de Marketing**: Incluye plantillas HTML para generar posts de Instagram coherentes con la marca.

## 🛠️ Tecnologías Utilizadas

*   **HTML5** Semántico.
*   **Tailwind CSS** (vía CDN) para un estilizado rápido y responsivo.
*   **JavaScript (Vanilla)** para la interactividad del cliente (modales, cálculos, tabs).

## � Adaptabilidad e Integración

Este diseño estático y modular está preparado para escalar. Su estructura limpia y basada en componentes facilita su migración a:

*   **CMS**: Puede convertirse fácilmente en un tema personalizado para **WordPress**.
*   **Frameworks Frontend**: Estructura lista para ser componentizada en **React**, **Angular** o **Vue.js**.
*   **Frameworks Backend**: Las plantillas HTML son compatibles con motores de templates como Jinja2 (**Django** / **Flask**) o Blade (**Laravel**).

## �🚀 Cómo visualizar el proyecto

Este es un sitio estático, por lo que no requiere instalación de dependencias ni servidores complejos.

1.  Clona este repositorio o descarga los archivos.
2.  Abre el archivo `index.html` en tu navegador web.
